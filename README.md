# NeuralMatrix

A minimalist, high-fidelity developer portfolio with a "Digital Matrix" theme.

## Features

- **Matrix Rain**: Performance-optimized HTML5 Canvas effect.
- **CRT Overlay**: Authentic scanline and screen curvature effects.
- **Terminal Typography**: Uses `Space Mono` and `VT323`.
- **Interactive Grid**: Project cards with staggered animations.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
